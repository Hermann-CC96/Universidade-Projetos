# 💻 Algoritmos e Programação

Repositório dedicado ao estudo de lógica de programação, estruturas de dados básicas e resolução de problemas computacionais.

## 🚀 O que foi desenvolvido
* **Lógica Proporcional:** Estruturas condicionais e de repetição.
* **Manipulação de Dados:** Vetores (arrays), Matrizes e Strings.
* **Modularização:** Criação de funções e procedimentos.
* **Estruturas Avançadas:** Ponteiros e alocação dinâmica de memória.

## 🛠 Linguagens Utilizadas
* 'C / C++', 'Python', ' JavaScript'
